# pedra-papel-tesoura

`by Hapistello Gnosko`

</div>

---

Pedra, papel e tesoura contra a máquina. Placar acumulado ao longo das rodadas. Primeiro projeto com loop de jogo e aleatoriedade controlada.

---

## `// tech`

![Python](https://img.shields.io/badge/Python-0d0d0d?style=for-the-badge&logo=python&logoColor=6A0DAD)

---

## `// usage`

```bash
git clone https://github.com/HapistelloGnosko/pedra-papel-tesoura
cd pedra-papel-tesoura
python jogo.py
```

---

## `// features`

- Escolha interativa: pedra, papel ou tesoura
- Decisão aleatória da máquina com `random`
- Placar acumulado por sessão
- Loop com opção de continuar ou encerrar
